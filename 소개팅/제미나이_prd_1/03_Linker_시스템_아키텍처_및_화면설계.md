# Linker(가칭) 시스템 아키텍처 및 UI/UX 화면 설계서

**문서 목적:** 하이엔드 지인 소개 매칭 웹 플랫폼 'Linker'의 초기(MVP) 개발을 위한 기술 스택(Tech Stack)을 확정하고, 핵심 유저 동선(User Flow) 및 화면(UI/UX) 설계를 정리합니다.

## 1. 추천 기술 스택 (Tech Stack)

### 1.1. 프론트엔드 (Frontend)
- **프레임워크:** Next.js(React 기반) - 초기 로딩 속도가 빠르고 SEO에 유리하여 웹 검색 유입에 좋음.
- **디자인/스타일링:** Tailwind CSS - 빠르고 세련된 반응형 웹 구축에 최적.

### 1.2. 백엔드 및 데이터베이스
- **서버 프레임워크:** Node.js(NestJS) 또는 Spring Boot(Java) - 빠른 런칭은 Node.js, 금융권 수준 트랜잭션 안정성은 Spring Boot.
- **데이터베이스:** PostgreSQL - 예치금 결제/정산/환불 등 데이터 무결성이 필수인 관계형 DB.

### 1.3. 외부 API 연동
- **본인 인증:** 토스(Toss) 본인확인 API
- **결제 및 에스크로:** 토스페이먼츠 가상계좌 및 파트너 정산 API
- **알림톡:** 카카오 알림톡 API(매칭 제안, 결제 완료, 만남 인증 요청 등)

## 2. 핵심 유저 플로우 (Core User Flow)

Linker는 '주선자'와 '매칭 희망자'의 진입 동선이 완전히 분리되어야 합니다.

### Flow A. 주선자(리셀러) 동선
1. 랜딩 페이지: "지인 소개해주고 성사 시 100만원 받으세요" 배너 클릭
2. 회원 가입: 토스 본인인증 → 주선자 약관 동의 → 정산 계좌 등록
3. 지인 등록(대시보드): 새로운 지인 프로필 올리기 → 신상(나이, 직업군, 거주지, 특징) 텍스트 작성 → 등록 완료
4. 매칭 관리: 시스템이 추천해준 매칭 희망자에게 내 지인의 프로필 제안
5. 정산 확인: 1회차 만남 성사(1~4 수수료 즉시 입금) / 3회차 커플 성사(70~120만원 입금) 내역 확인 및 출금 신청

### Flow B. 매칭 희망자(Seeker) 동선
1. 랜딩 페이지: "알바 없는 100% 실 지인 소개, 성사되지 않으면 잔금 결제 없음" 배너 클릭
2. 회원 가입: 토스 본인인증 → 예치금(Hold) 환불 규정 필수 동의
3. 프로필 수신: 주선자들이 제안한 프로필 도착 알림톡 수신
4. 수락 및 결제: 프로필 확인 후 만남 수락 클릭 → 가상계좌로 초기 결제(예치금 5 + 1회차 주선비 1) 입금
5. 만남 인증: 오프라인 만남 후 시스템에 1회차 만남 완료 버튼 클릭(주선자와 교차 검증) → 3회차 완료 시 잔금 결제 페이지로 이동

## 3. UI/UX 화면 설계(주요 와이어프레임 컨셉)

기존 데이팅 앱처럼 화려하고 가벼운 핑크빛 디자인은 배제합니다. 고급스러운 프라이빗 라운지, 헤드헌팅 서비스, 블랙&골드 또는 딥 네이비 톤의 신뢰감 있는 디자인이 필요합니다.

### 3.1. 메인 홈 화면(Landing Page)
- 상단 헤더: 로고(Linker) / 주선자로 활동하기 버튼 / 소개받기 버튼 / 로그인
- 메인 히어로 섹션: 카피 "검증된 지인들의 프라이빗 연결, 가벼운 만남은 사양합니다." / 시각 자료: 프라이빗한 카페/라운지 느낌의 고해상도 배경
- 신뢰 강조 섹션: "왜 Linker인가?" - 1. 무조건 실 지인 주선(알바 0%) / 2. 만남 성사 시에만 결제 / 3. 가벼운 유저 100% 필터링
- 하단 콜투액션 듀얼 버튼: [내 지인 소개하고 수익 창출하기(주선자용)] / [진짜 인연 소개받기(희망자용)]

### 3.2. 매칭 희망자 대시보드(프로필 제안 확인 화면)
- 도착한 제안 리스트: 나에게 프로필을 보낸 리스트가 블라인드 카드 형태로 나열됨.
- 카드 상세 보기: 주선자의 추천사("제 직장 후배입니다. 성실하고 예의가 바릅니다." - 주선자 신뢰도 별점 표시), 상대방 기본 정보(1993년생 / 서울 강남구 / IT 기획자, 사진은 초기엔 흐리게 처리되거나 상호 동의 후 공개), 버튼: [이 분과 커피 한 잔하기(결제 6 진행)] / [거절하기]

### 3.3. 주선자 대시보드(리셀러 센터)
- 상단 요약 현황판: 총 주선 횟수 5건 / 진행 중인 만남 2건 / 커플 성사 1건, 누적 출금 가능 금액 1,250,000원 [출금하기 버튼]
- 내 지인 리스트 관리: 내가 등록한 지인들의 프로필 카드와 현재 누구와 매칭 진행 중인지 상태(1회차, 2회차, 성사 대기 등) 표시.
