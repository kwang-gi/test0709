# 요약: 길게 (Long)

## 역할
가장 촘촘한 요약. 소제목 아래 세부 항목(a, b, c)까지 전개.

## 규칙
- 스크립트 안에서만 생성한다.
- 소제목(##) 3~7개, 각 소제목 아래 번호 목록(1,2,3) + 하위 a,b,c 항목까지 허용.
- 예시, 부연 설명, 배경 맥락까지 포함해서 자세히 쓴다.
- 각 항목 끝에 근거 타임스탬프 [mm:ss] 표기.
- 스크립트에 언급된 순서를 최대한 따른다.

## 출력 형식
```
# (영상 제목) - 상세 요약

## 1. 소제목 [mm:ss]
1. 주요 내용 [mm:ss]
   a. 세부 설명 [mm:ss]
   b. 세부 설명/예시 [mm:ss]
2. 주요 내용 [mm:ss]

## 2. 소제목 [mm:ss]
...
```

## 프롬프트 (LLM에 그대로 전달)
"아래 스크립트를 아주 자세하게, 소제목 3~7개와 그 아래 번호 목록 + 하위 세부항목(a,b,c)까지 전개해서 요약해줘. 예시나 부연설명도 포함하고, 각 항목 끝에 근거 타임스탬프를 [mm:ss]로 붙여."
