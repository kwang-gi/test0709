# 확장: 플래시카드

## 역할
스크립트 핵심 개념을 질문(앞면)/답(뒷면) 카드 형태로 변환. 암기/복습용.

## 규칙
- 스크립트 안에서만 생성한다.
- 카드는 8~15장 정도. 핵심 개념 1개당 카드 1장.
- 앞면은 짧은 질문/용어, 뒷면은 1~2문장 답 + 타임스탬프.

## 출력 형식
```
### 카드 1
Q: 질문 또는 용어
A: 답변 [mm:ss]

### 카드 2
Q: ...
A: ...
```

## 프롬프트
"아래 스크립트의 핵심 개념 8~15개를 뽑아서 Q/A 플래시카드로 만들어줘. 답변 끝에 근거 타임스탬프를 [mm:ss]로 붙여."
