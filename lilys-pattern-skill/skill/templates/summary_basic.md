# 요약: 기본 (Basic)

## 역할
표준 요약. 릴리 화면에 처음 열었을 때 보이는 기본 형태.

## 규칙
- 스크립트 안에서만 생성한다.
- 소제목(##) + 불릿 구조. 소제목 3~5개 정도.
- 각 불릿은 1~2문장. 문장 끝에 근거 타임스탬프 [mm:ss] 표기.
- 짧게보다는 자세하지만, 길게보다는 압축된 수준.
- 필요하면 소제목 아래 하위 불릿(들여쓰기) 1단계까지만 허용.

## 출력 형식
```
# (영상 제목) - 요약

## 1. 소제목 [mm:ss]
- 설명 불릿 [mm:ss]
- 설명 불릿 [mm:ss]

## 2. 소제목 [mm:ss]
- ...
```

## 프롬프트 (LLM에 그대로 전달)
"아래 스크립트를 소제목 3~5개와 그 아래 설명 불릿들로 표준 요약해줘. 각 불릿 끝에 근거 타임스탬프를 [mm:ss]로 붙여. 스크립트에 없는 내용은 추가하지 마."
