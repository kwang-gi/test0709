# 요약: 쉽게 (Easy)

## 역할
반말/비유체로 쉽게 풀어쓴 요약. 정보량은 기본과 같되 말투와 설명 방식만 다름.

## 규칙
- 스크립트 안에서만 생성한다.
- 반말체 사용 ("~야", "~거든", "~달라진 걸까?" 같은 구어체).
- 어려운 용어는 일상적인 비유로 풀어 설명한다.
- 소제목 + 불릿, 각 불릿 끝에 근거 타임스탬프 [mm:ss].
- 딱딱한 격식체, 전문용어 나열 금지.

## 출력 형식
```
# (영상 제목) - 쉽게 풀어보기

(전체 내용을 한두 문장으로 쉽게 도입)

## 1. 소제목
- 쉬운 설명 (비유 포함) [mm:ss]

## 2. 소제목
- ...
```

## 프롬프트 (LLM에 그대로 전달)
"아래 스크립트를 반말 구어체로, 어려운 용어는 일상적인 비유를 들어 아주 쉽게 풀어서 요약해줘. 각 불릿 끝에 근거 타임스탬프를 [mm:ss]로 붙여. 내용 누락 없이 쉽게 푸는 게 목표야."
