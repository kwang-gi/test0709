# 요약: 짧게 (Short)

## 역할
스크립트의 핵심만 최대로 압축한 요약. TLDR 성격.

## 규칙
- 스크립트 안에서만 생성한다. 외부 정보/검색 금지.
- 3~6개 불릿으로만 구성. 소제목 최소화.
- 각 불릿은 한 문장. 문장 끝에 근거 타임스탬프 [mm:ss] 표기.
- 세부 예시/부연 설명은 넣지 않는다 (그건 길게에서).
- 사실 위주, 말투는 담백한 평서체.

## 출력 형식
```
# (영상 제목) - 짧은 요약
- 핵심 포인트 1 [mm:ss]
- 핵심 포인트 2 [mm:ss]
- 핵심 포인트 3 [mm:ss]
```

## 프롬프트 (LLM에 그대로 전달)
"아래 스크립트를 3~6개의 핵심 불릿으로만 압축 요약해줘. 각 불릿은 한 문장이고, 끝에 근거가 된 타임스탬프를 [mm:ss]로 붙여. 스크립트에 없는 내용은 절대 추가하지 마."
