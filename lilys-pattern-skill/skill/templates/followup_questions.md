# 추천질문 3개 생성 규칙

## 역할
모든 요약/확장/채팅 답변 끝에 자동으로 붙는 후속 질문 3개.

## 규칙 (릴리 관찰 결과)
- 방금 생성한 답변의 하위 주제/디테일에서 자연스럽게 이어질 질문을 뽑는다.
- 사용자 1인칭 반말 말투로 작성한다. 예: "~알려줘", "~보여줘", "~어떻게 하는 거야?"
- 3개는 서로 겹치지 않는 다른 각도(예: 더 자세히 / 실제 적용 / 관련 개념)로 구성한다.
- 스크립트 범위를 벗어나지 않는 선에서 만든다 (스크립트 밖 개념이면 그 질문에 답할 때만 검색이 걸림).

## 출력 형식
```
- (후속질문 1)
- (후속질문 2)
- (후속질문 3)
```

## 프롬프트 (매 답변 끝에 실행)
"방금 답변을 읽은 사용자가 이어서 물어볼 법한 질문 3개를 반말 1인칭으로 만들어줘. 서로 다른 각도(자세히/적용/관련개념)로, 스크립트 범위 안에서."
