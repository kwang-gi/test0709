# 확장: 커스텀 (나만의 템플릿)

## 역할
릴리의 "나만의 템플릿 만들기"와 동일한 기능. 사용자가 이름과 프롬프트만 정의하면 새 확장이 된다.

## 사용법
1. 이 파일을 복제해서 이름을 바꾼다 (예: ext_회의록.md).
2. 아래 두 항목만 채운다.

## 템플릿 정의
- 이름: (예: 회의록 형식으로 정리)
- 프롬프트: (예: "아래 스크립트를 회의록 형식(참석자/안건/결정사항/액션아이템)으로 정리해줘. 근거 타임스탬프[mm:ss]도 붙여.")

## 규칙 (모든 커스텀 템플릿 공통)
- 기본은 스크립트 안에서만 생성 (검색 필요하면 프롬프트에 명시).
- 결과에 근거 타임스탬프 [mm:ss] 포함.
- 답변 끝에 추천질문 3개 자동 첨부 (followup_questions.md 규칙 따름).

## 예시 (이미 정의된 커스텀 하나)
- 이름: 팟캐스트 대본
- 프롬프트: "아래 스크립트를 2명이 대화하는 팟캐스트 대본 형식으로 재구성해줘. 자연스러운 구어체로, 핵심 내용은 누락하지 마."
