# 확장: 반대 시각 조사 (웹검색 사용)

## 역할
스크립트 주장에 대한 반박, 한계, 비판적 시각을 조사해서 균형 잡힌 시각 제공.

## 규칙
- 스크립트의 핵심 주장을 먼저 1~2줄로 요약한다.
- 그 주장에 대한 반대/비판적 시각을 웹검색으로 찾는다.
- 스크립트에 이미 반박이 있으면 그것부터 쓰고, 부족하면 웹검색으로 보충한다.
- 웹 출처는 반드시 도메인으로 표기한다.

## 출력 형식
```
## 반대 시각 조사

### 원 주장
- (스크립트 주장) [mm:ss]

### 반대/비판적 시각
- 반박 포인트 1 (출처: example.com)
- 반박 포인트 2 (출처: example.com)

### 균형 잡힌 결론
- 정리 문장
```

## 프롬프트
"스크립트의 핵심 주장을 요약하고, 그에 대한 반박/비판적 시각을 웹검색으로 찾아줘. 출처 도메인을 꼭 표기하고, 마지막에 균형 잡힌 결론을 덧붙여."
