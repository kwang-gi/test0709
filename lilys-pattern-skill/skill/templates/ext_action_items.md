# 확장: 내 액션아이템 도출

## 역할
스크립트 내용을 실제로 실행할 수 있는 "왜 해야 하나요 / 어떻게 하나요 / 실행 체크리스트" 형태로 변환.

## 규칙
- 스크립트 안에서만 생성 (검색 없음).
- 영상에서 다룬 각 주요 주제마다 아래 3단 구조를 반복한다.
- 체크리스트는 실제로 따라할 수 있는 순서형 행동으로 작성한다.

## 출력 형식
```
## (주제 제목)

### 왜 해야 하나요?
- 이유 1 [mm:ss]
- 이유 2 [mm:ss]

### 어떻게 하나요?
1. 단계 1 [mm:ss]
2. 단계 2 [mm:ss]

### 실행 체크리스트
- [ ] 할 일 1
- [ ] 할 일 2
```

## 프롬프트
"아래 스크립트에서 다루는 각 주요 주제마다, 왜 해야 하는지/어떻게 하는지/실행 체크리스트 형태로 액션아이템을 뽑아줘. 스크립트에 실제로 언급된 내용만 사용하고 타임스탬프를 [mm:ss]로 표기해."
