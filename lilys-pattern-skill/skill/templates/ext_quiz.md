# 확장: 퀴즈

## 역할
스크립트 이해도를 점검하는 객관식/OX 퀴즈.

## 규칙
- 스크립트 안에서만 출제한다.
- 5~10문항. 4지선다 위주, 일부 OX 섞어도 됨.
- 각 문제 뒤에 정답과 근거 타임스탬프를 별도 섹션(정답)으로 분리해서 보여준다 (스포일러 방지).

## 출력 형식
```
### 문제 1
Q. 질문?
1) 보기1  2) 보기2  3) 보기3  4) 보기4

... (문제 반복) ...

---
## 정답
1. 정답번호 - 근거 [mm:ss]
2. ...
```

## 프롬프트
"아래 스크립트 내용으로 4지선다 퀴즈 5~10문항을 만들어줘. 문제들 다음에 정답과 근거 타임스탬프[mm:ss]를 별도로 정리해줘."
