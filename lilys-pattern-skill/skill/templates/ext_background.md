# 확장: 관련 배경지식 (웹검색 사용)

## 역할
스크립트에 언급되지만 자세히 설명되지 않은 용어/고유명사/맥락을 외부 웹검색으로 보충 설명.

## 규칙 (릴리 관찰 결과 반영)
- 먼저 스크립트만으로 답이 되는지 확인한다. 스크립트 안에 이미 있으면 검색하지 않는다.
- 스크립트에 없는 새 외부 개념/고유명사가 나올 때만 웹검색을 수행한다 (1회).
- 검색 결과의 출처(도메인)를 반드시 표기한다.
- 스크립트 기반 부분은 [mm:ss], 웹 기반 부분은 (출처: 도메인) 으로 구분 표기한다.

## 출력 형식
```
## 관련 배경지식: (용어/개념)

- 스크립트에서 언급된 내용: ... [mm:ss]
- 추가 배경(웹): ... (출처: example.com)
```

## 프롬프트
"스크립트에서 자세히 설명되지 않은 핵심 용어나 고유명사를 찾아줘. 스크립트 안에서 설명 가능하면 검색하지 말고, 정말 스크립트에 없는 개념일 때만 웹검색으로 보충하고 출처를 표기해."
